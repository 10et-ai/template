# TENET Workspace

Read `CLAUDE.md` for full project instructions.

## Quick Start

1. Use `tenet_context` to get project state
2. Use `tenet_memory_search` to find past decisions
3. Write journal entries as you work (see CLAUDE.md)

## Available Tools

- **tenet_context** — project context, journal, knowledge docs
- **tenet_memory_search** — search past decisions and work
- **tenet_memory_add** — persist decisions to project memory
- **tenet_hud** — project dashboard
- **tenet_pivot** — checkpoint work mid-session
- **tenet_synopsis** — summarize recent work

## Rules

- **Journal every significant action** — features, decisions, fixes, discoveries
- **Every code file needs `@purpose` header**
